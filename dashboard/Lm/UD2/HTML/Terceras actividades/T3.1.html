  1. alt: Texto alternativo en caso de que la imagen no se pueda mostrar.
  2. coords: Conjunto de valores que epecifican las coordenadas del area.
  3. download: Indica que el hipervínculo es utilizado para descargar el recurso.
  4. href: La URL de un recurso asociado.
  5. hreflang: Especifica el lenguaje del recurso asociado.
  6. media: Da un inidcio del tipo para el cual el recurso vinculado fue creado.
  7. ping: Indica si se enviara un ping o notificación a la URL del recurso cuando el usuario utilice el vínculo.
  8. rel: Especifica la relación entre el objeto destino y el objeto enlace.
  9. shape: Define una región seleccionable para hacer hipervínculos asociados con una forma asoaciada a una figura qpara ccrear un mapa. Los valores para este atributo son: circle, defaul, plygon y rect.
  10. target: Especifica el marco destino en un vínculo. Valores especiales: "_blank" indica una nueva ventana, "_parent" indica el marco padre del marco que contiene el código fuente, "_self" indica el marco donde está el código fuente y "_top" indica la ventana completa del navegador.
