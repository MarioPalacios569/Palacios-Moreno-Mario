Actividad 2 ¿Para que se utilizan los elementos map y area ?

Respuesta

El elemento map se utiliza para crear una caja en bloque definido como elemento especial.

El elemento area se utiliza para crear las distintas ssecciones en las que se puede dividir un map. Hay que poner los siguientes valores

  1. Shape: Forma del area, sus valores son: rect, circle, poly, default
  2. Coords: Detalla las coordenadas del atributo shape
  3. Href: Crea hipervínculo para el área, es decir, se redirige al dar click a la figura
  4. Alt: Texto alternativo
